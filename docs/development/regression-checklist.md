# 交互回归清单

> 大改 UI、拖拽、AI、图片或数据维护前后各跑一遍。自动验证先执行 `bash scripts/verify-local.sh`。

## 基础启动

- [ ] `docker compose ps` 显示 frontend / backend 都是 Up。
- [ ] `http://localhost:5173` 能打开当前面板。
- [ ] `http://localhost:8000/api/health` 返回 `{"status":"ok"}`。

## 任务面板

- [ ] 当前 / 待 Review / 归档 三段切换数量和列表一致。
- [ ] 搜索标题、备注、状态、日期能过滤任务，清空筛选后列表恢复。
- [ ] 数字键 `1` / `2` / `3` 能切换三段视图，`4` 能切换象限 / 坐标布局。
- [ ] `[` / `]` 能切换日期，历史日期不能新建任务。
- [ ] 任务较多时整页只有一个滚动条，象限内部不再单独滚动；向下滚动顶栏吸顶常驻。
- [ ] 卡片备注完整展开，不被截断为两行。

## 拖拽和排序

- [ ] 当前视图、无筛选时，同象限拖拽能改变顺序。
- [ ] 拖到左列会清空截止日期。
- [ ] 拖到有期限象限会保持或继承合理截止日期。
- [ ] 筛选中不会误允许拖拽排序。

## 编辑弹窗

- [ ] 新建任务默认今天、重要、待办。
- [ ] 编辑任务后 `Ctrl/Cmd + Enter` 保存，Esc 触发关闭流程。
- [ ] 有未保存内容时点弹窗外，会出现保存 / 不保存 / 继续编辑确认。
- [ ] 删除任务必须二次确认。

## AI 草稿

- [ ] AI 输入框 Enter 可以拆任务，成功后逐条弹出预填编辑窗。
- [ ] AI 快捷模板和最近输入按钮能填回输入框。
- [ ] 草稿状态能识别待 Review / 待验证 / 进行中 / 已完成。
- [ ] 保存草稿才会入库，关闭草稿不会新增任务。
- [ ] AI 输入区能一键收起 / 展开，刷新后保持上次状态；周回顾「填入 AI 总结」或当前筛选复盘会自动展开输入区。

## 图片

- [ ] 编辑弹窗粘贴图片后有缩略图。
- [ ] 卡片缩略图点击会复制图片。
- [ ] 弹窗点图进入灯箱，复制按钮可用；右键菜单不被拦截。
- [ ] 删除任务后关联图片文件不会继续占用维护概览。

## 维护和备份

- [ ] 备份弹窗能显示任务数、图片数、数据体积。
- [ ] 图片健康正常时显示文件一致。
- [ ] `curl http://localhost:8000/api/maintenance/summary` 包含孤儿和缺失图片统计。
- [ ] 顶部导出生成的 Markdown 包含当前日期、视图、筛选条件和任务明细。

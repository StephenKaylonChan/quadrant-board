## 代码事实摘要

- 后端是 FastAPI + async SQLAlchemy + SQLite，入口在 `backend/app/main.py`。
- 数据库文件和上传图片都在 `data/`，Compose 把宿主机 `./data` 挂到容器 `/code/data`。
- 任务主模型是 `important + due_date`；旧 `urgency / importance` 只保留兼容老库。
- 每日面板由 `created_date <= D 且 (completed_date 为空 或 >= D)` 查询得到，没有每日快照表。
- 状态为 `todo / doing / review / done`，只有 `done` 写入 `completed_date`。
- 前端是 React 19 + Vite 6 + TypeScript，无 UI 库，单屏应用，没有路由库和状态库。
- Vite 代理 `/api` 和 `/uploads` 到后端容器 `backend:8000`。
- AI 拆任务走 OpenAI 兼容 `/chat/completions`，只返回草稿，不直接入库。

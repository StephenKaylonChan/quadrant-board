# Changelog

## [Unreleased]

### Added

- 初始化 Claude Code 协作系统文档。
- 生成 architecture / development 文档生态守护报告。
- 新增项目背景文档，记录工作流痛点、Will 建议和任务模型演进。
- 补充 GitHub 仓库、`origin/main`、clone、回滚和容器内构建验证说明。
- 新增 `smoke:api` 基础接口冒烟检查，覆盖任务创建、完成日期、恢复和删除链路。
- 新增任务搜索和范围筛选工具栏，支持按标题、备注、状态、日期检索，并在筛选中保护拖拽排序。
- 新增当前面板 Markdown 导出，导出内容会跟随当前日期、视图、搜索词和范围筛选。
- 新增周回顾面板，统计最近 7 天新增、完成、当前待处理、待 Review、待验证和过期任务，并支持复制回顾。
- 新增备份说明弹窗，说明 Markdown 轻量导出和 `data/` 完整备份边界。
- 新增可点击当日概览筛选，支持过期、今日截止、明日截止、隔夜进行等重点视图。
- 新增全局键盘快捷键：`N` 新建任务、`[` / `]` 切换日期、`T` 回到今天。
- 新增收口建议条，自动聚合过期、今日截止、待验证、隔夜进行等优先任务。
- 新增筛选条件标签，可单独移除搜索、范围、状态和重点筛选。
- 新增任务卡片快速操作，支持复制任务文本和快速流转状态。
- 新增视图记忆，刷新后保留上次使用的当前 / 待 Review / 归档视图。
- 新增任务卡片期限快捷操作，支持一键改到今天、明天或清空期限。
- 新增数字键 `1` / `2` / `3` 切换任务视图。
- 新增压力坐标视图，支持在当前 / 待 Review / 归档和筛选结果中按重要性、时限压力查看任务分布。
- 新增面板布局记忆，支持在象限视图和坐标视图之间切换并保留上次选择。
- 新增 `?layout=scatter` / `?layout=quadrant` 直达布局参数，便于回归截图和固定打开方式。
- 新增数字键 `4` 快速切换象限 / 坐标布局。
- 新增 AI 输入快捷模板和最近输入记忆，减少重复描述常见拆解场景。
- 新增收口建议首项快捷打开按钮，并支持键盘 `F` 聚焦第一条建议任务。
- 新增前端错误边界，页面渲染异常时显示可恢复的错误面板而不是直接白屏。
- 新增只读数据维护概览接口，并在备份弹窗中展示任务数、图片数和数据体积。
- 新增 AI 解析轻量回归测试，覆盖代码块 JSON、字符串布尔值、异常状态和草稿数量上限。
- 新增全局按钮键盘焦点样式和减少动态效果偏好支持，改善键盘导航与低动效体验。
- 新增上传图片健康检查，维护概览会统计孤儿图片文件和数据库记录缺失的图片文件。
- 优化备份弹窗的数据概览，直接展示图片文件是否一致以及异常样例。
- 新增本地交付验证脚本和交互回归清单，覆盖构建、冒烟、后端测试、拖拽、AI 草稿和图片复制。
- 新增后端 live API 回归测试，覆盖任务创建、日期窗口、完成归档、恢复和删除。
- 新增数据库轻量迁移策略文档，明确 `init_db()` 补列模式、适用边界和执行前检查。
- 新增 `data/` 备份和恢复脚本，并补充可恢复备份目录要求和恢复后验证流程。
- 新增孤儿图片清理脚本，默认 dry-run，显式传 `--apply` 才会删除未被数据库引用的上传文件。
- 新增当前视图 JSON 数据包导出，包含格式版本、筛选条件、任务字段和图片文件引用。
- 优化 AI 草稿队列，弹窗上方显示当前进度和剩余数量，并支持一键跳过全部草稿。
- 新增维护接口拆分：`upload-health` 单独返回图片一致性，`cleanup-preview` 返回清理 dry-run 结果。
- 优化移动端顶部操作、横向筛选和坐标视图触控尺寸，减少首屏被工具栏占满的问题。
- 新增前端轻量单元测试，覆盖视图筛选、当日概览、收口建议和 JSON 导出格式。
- 新增剪贴板回归测试，覆盖权限拒绝兜底提示和非浏览器环境的不支持提示。
- 新增 AI 拆任务去重上下文,请求会携带当前未归档任务标题供模型参考。
- 优化 AI 拆解失败状态,错误提示旁提供重试和清空操作。

- 优化任务编辑器保存体验，标题长度与后端限制对齐，并支持 `Ctrl/Cmd+S` 保存。
- 优化今日同步和 Markdown 导出，补充收口建议段落。
- 优化周回顾，展示本周净变化和每日待处理数量。

### Fixed

- 修复文档中仍把有期限象限描述为状态优先排序的问题。
- 修复部分浏览器拒绝灯箱程序化复制图片时缺少原生右键兜底的问题。
- 修复粘贴包含非图片文件时会误拦截普通粘贴的问题，并提前提示超过 10MB 的图片。
- 修复 AI 返回字符串 `"false"` 时会被误判为重要任务的问题，并限制异常超长草稿数量。
- 修复剪贴板工具在 SSR / Node 测试环境访问 `navigator` 时可能抛底层引用错误的问题。

# Changelog

## [Unreleased]

### Added

- 初始化 Claude Code 协作系统文档。
- 生成 architecture / development 文档生态守护报告。
- 新增项目背景文档，记录工作流痛点、Will 建议和任务模型演进。
- 补充 GitHub 仓库、`origin/main`、clone、回滚和容器内构建验证说明。
- 新增 `smoke:api` 基础接口冒烟检查，覆盖任务创建、完成日期、恢复和删除链路。
- 新增任务搜索和范围筛选工具栏，支持按标题、备注、状态、日期检索，并在筛选中保护拖拽排序。
- 新增当前面板 Markdown 导出，导出内容会跟随当前日期、视图、搜索词和范围筛选。
- 新增周回顾面板，统计最近 7 天新增、完成、当前待处理、待 Review、待验证和过期任务，并支持复制回顾。
- 新增备份说明弹窗，说明 Markdown 轻量导出和 `data/` 完整备份边界。
- 新增可点击当日概览筛选，支持过期、今日截止、明日截止、隔夜进行等重点视图。
- 新增全局键盘快捷键：`N` 新建任务、`[` / `]` 切换日期、`T` 回到今天。

### Changed

- 同步 AGENTS、CLAUDE、README、架构文档、上手指南和部署说明到当前代码事实。
- 标记路线图中 AGENTS 旧模型说明修正项为完成。
- 同步有期限象限的日期优先排序规则、五档状态和图片复制交互说明。
- 将面板展示改为「当前 / 待 Review / 归档」三段视图，Review 和归档复用完整卡片展示。
- 优化任务卡片信息密度，显示创建日期和图片数量。
- 优化任务编辑器保存体验，标题长度与后端限制对齐，并支持 `Ctrl/Cmd+S` 保存。

### Fixed

- 修复文档中仍把有期限象限描述为状态优先排序的问题。
- 修复部分浏览器拒绝灯箱程序化复制图片时缺少原生右键兜底的问题。
- 修复粘贴包含非图片文件时会误拦截普通粘贴的问题，并提前提示超过 10MB 的图片。
